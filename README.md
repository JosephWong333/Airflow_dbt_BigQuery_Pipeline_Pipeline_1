This project is an end-to-end data pipeline that uses Airflow to orchestrate loading data into BigQuery, dbt for transformations, and Soda for quality checks. It is built using the Astronomer CLI, providing a containerized environment with all the necessary folders for DAGs, requirements, and configuration files. To get started, you simply run astro dev start to launch the Airflow components locally via Docker and access the UI at port 8080.
